# Installation
### Download and change folder
```
cd ~
git clone https://github.com/flipsidecreations/dotfiles.git

cd dotfiles
```
My version is now

```
git clone https://github.com/surfermc/dotfiles.git
```

### Run install
```
./install.sh
```

# Uninstallation

### Change to dotfiles folder
```
cd ~/dotfiles
```

### Run uninstaller
```
./uninstall.sh
```
